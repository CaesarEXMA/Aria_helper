# Install
1. [Python 3.x](https://python.org)
2. for windows u need install [VC++ build tools](https://visualstudio.microsoft.com/downloads) 2015 or Latest Version
3. [Rust](https://rust-lang.org)

```
pip install -r requirements.txt
```
